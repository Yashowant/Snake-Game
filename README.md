# Snake-Game
To make a Snake game using python For beginner.

How to download and run the file:-

Step-1

Download the .py file

Step-2

Use an IDE to open the code, such as CMD, Powershell, and even VS Code

Step-3

install the module:-

pip install Random
pip install Time
pip install PyGame

Step-4

Run this code in Terminal.
